# Global Unicorn Companies Insights Dashboard

## Project Overview
This project analyzes global unicorn companies to understand funding trends, valuation distribution, industry performance, and geographic expansion using Power BI.

## Tools Used
- Power BI
- Power Query
- Data Visualization
- Excel / CSV datasets

## Dataset
The analysis uses datasets containing information about unicorn companies, funding rounds, investors, and company valuations.

## Dashboard Features
- Funding vs Valuation Analysis
- Industry-wise Unicorn Distribution
- Geographic Distribution of Unicorn Companies
- Investor Insights
- KPI Cards for Funding, Valuation, and Company Count

## Key Insights
- North America contains the largest share of unicorn companies.
- FinTech and Internet industries dominate unicorn startup growth.
- Significant variation exists between funding raised and company valuation.
